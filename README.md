# Gaurav-99.github.io

Personal static portfolio / resume website built with HTML, CSS and JavaScript.

## Description

This repository hosts a single-page personal website that showcases a resume and a small interactive particle background demo.

## Features

- Static HTML/CSS/JS (no build step required)
- Animated particle background using the jQuery `particleground` plugin (see `js/`)
- Resume available in the `Resume/` folder

## Repository structure

- `index.html` — main page
- `assets/` — images and other assets
- `css/style.css` — site styles
- `js/` — JavaScript files (`demo.js`, `jquery.particleground.js`, `jquery.particleground.min.js`)
- `Resume/` — resume files
